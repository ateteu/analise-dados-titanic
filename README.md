# Titanic Data Analysis

Pequeno projeto de análise exploratória do dataset Titanic utilizando Python.

## Objetivo

Praticar manipulação e exploração de dados com Pandas, incluindo:

- carregamento de dados
- inspeção do dataset
- tratamento de valores nulos
- agrupamentos e agregações
- visualização simples

## Dataset

Titanic passenger dataset.

Fonte:
[https://github.com/datasciencedojo/datasets](https://raw.githubusercontent.com/datasciencedojo/datasets/master/titanic.csv)

## Ferramentas utilizadas

- Python
- Pandas
- Matplotlib
- Jupyter Notebook
